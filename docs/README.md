# English Lesson 项目文档

本目录用于存放工程实施相关文档，避免将课程规范与开发计划混在同一个文件中。

## 文档索引

- [`../lesson_design.md`](../lesson_design.md)：课程规范，定义课程目标、课程结构、教学原则、复习机制、课程数据模型与 V1 范围边界。
- [`development-plan.md`](development-plan.md)：完整开发计划，定义技术方案、页面结构、文件结构、音频策略、进度存储、录音回放、开发阶段与验证方式。
- [`minimum-slice-plan.md`](minimum-slice-plan.md)：首批最小垂直切片计划，定义 Day 0 与前三节课的最小可运行范围、功能清单、数据样例、执行步骤与验收标准。

## 建议阅读顺序

1. 先读 [`../lesson_design.md`](../lesson_design.md)，确认课程目标、课程边界与教学原则。
2. 再读 [`development-plan.md`](development-plan.md)，理解完整前端实现方案。
3. 最后按 [`minimum-slice-plan.md`](minimum-slice-plan.md) 执行首批最小切片开发。
