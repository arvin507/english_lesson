# English Zero（EZP）V1.2 课程规范（Curriculum Spec）
> 面向 7~10 岁零基础儿童（2年级暑假 → 3年级起点）
> 形式：静态 HTML，孩子自主学习，家长精力有限，不报班
> 目标：8周建立"可拼读 + 可阅读 + 可跟课"基础能力

---

# 0. 项目背景与约束（新增）

- 孩子：小学2年级暑假，此前上过英语课但基础可视为0
- 家长：白天工作，无法逐课陪伴，只能做初期启动引导
- 产品形式：**静态 HTML**，无后端、无账号系统
- **不做**：自动判分、打卡激励、复杂进度可视化 —— 这些留给后续版本
- **要做**：用最低成本解决"孩子知道学到哪了"和"孩子知道自己读得对不对"这两个基本问题

---

# 1. 课程设计目标（V1 Only）

本版本只解决三个核心能力：

## 1.1 解码能力（Decoding）
孩子看到单词可以尝试读出来：cat / dog / pen / pig

## 1.2 基础阅读能力（Reading）
能读简单句：I am Tom. / This is a cat.

## 1.3 课堂听力能力（Listening）
能听懂基础课堂指令：Stand up. / Sit down. / Open your book.

---

# 2. 课程核心原则（Must Follow）

## 2.1 One Lesson One Focus（单一目标）
每节课只做一件事：学一个发音 / 一个拼读结构 / 一组阅读句子。禁止混合多个新知识点。

## 2.2 Input First（输入优先）
顺序必须是：**听 → 看 → 读 → 拼 → 说**
不允许：背单词优先 / 语法讲解优先

## 2.3 No Grammar Teaching（不教语法）
不讲主谓宾、时态、名词动词概念。只通过句子习得。

## 2.4 High Repetition（高重复）
同一词必须出现至少：当课3次 / 复习2次 / 阅读1次

## 2.5 Controlled Vocabulary（词汇控制）
每节课新词 ≤ 5个；阅读中陌生词 ≤ 20%

## 2.6 无判分、有参照（新增）
不做自动打分系统。每个新词/新句配一段标准发音音频，孩子听完自己模仿，再回放对比，靠孩子自己的耳朵纠音，不引入评判机制。

## 2.7 进度自动记忆，不需要孩子或家长规划（新增）
用浏览器本地存储（localStorage）记录"孩子学到第几课"。孩子打开页面自动跳转到应该学的那一课，不需要每次手动找进度，也不需要家长安排课表。

---

# 3. 课程结构（8 Weeks）

## Phase 1：声音系统（Week 1-2）
目标：建立"字母=发音"概念。内容：元音短音 / 基础辅音 / CVC结构

## Phase 2：拼读系统（Week 3-5）
目标：Word Family 拼读能力。内容：-at / -an / -og / -ip / -en，CVC扩展

## Phase 3：阅读系统（Week 6-7）
目标：短句阅读。内容：简单句型 / 高频词融合 / mini story

## Phase 4：校内衔接（Week 8）
目标：对接三年级课程。内容：Hello / greetings / classroom English / Unit 1-2 preview

> 备注：Phase 4 的课堂用语与前7周拼读体系关联较弱，建议后续版本考虑将部分课堂用语（如指令类高频词）提前分散渗透到 Phase 1-3 的"高频词系统"中，而不是集中在最后一周。

---

# 4. 每节课标准结构（Lesson Template）

每节课固定 20~25 分钟：

## 4.1 Review（3 min）
复习：昨天内容 / 3天前内容 / 高频词。形式：听 + 跟读。
> 具体复习哪一课的内容，由 localStorage 记录的进度自动决定（见第8节）。

## 4.2 New Sound（5 min）
引入唯一新知识，例如：/æ/ 或 -at

## 4.3 Word Practice（5 min）
3~5个单词：cat / hat / bat / mat
> 每个词配标准发音音频，孩子跟读后自己回放对比（见第7节）。

## 4.4 Reading（5 min）
2~4句短句：I see a cat. / The cat is big.

## 4.5 Practice Game（5 min）
固定模板：听音选图 / 拼词拖拽 / 找单词 / 跟读+回放对比

## 4.6 Summary（2 min）
今日完成提示（无星级评分，仅"完成"状态），点击"完成"后自动记录进度并解锁下一课。

---

# 5. Vocabulary Strategy（词汇策略）

## 5.1 不按主题学词
不做水果类/动物类/颜色类分类记忆，原因：不利于拼读训练

## 5.2 按"拼读规律"组织
例如：-at / -an / -og / -ip / -en

## 5.3 高频词单独系统
不混入拼读课，例如：I / am / is / the / my / you

---

# 6. Reading System（阅读系统）

## 6.1 三原则
可解码（80%已学过）/ 短句优先 / 高频重复

## 6.2 阅读结构
- Level 0：A cat.
- Level 1：I see a cat.
- Level 2：The cat is big.
- Level 3：I have a cat. It is black.

---

# 7. 反馈机制（原 Game System，已简化）

不引入判分逻辑，只提供"参照 + 自己对比"：

## 7.1 听音选图
听标准音频，选对应图片，页面直接显示正确答案（无评分，只有对错提示）

## 7.2 拼词拖拽
拖拽字母组词，完成后显示正确拼法作对照

## 7.3 找单词
在句子中找出学过的词，点击后高亮显示

## 7.4 跟读 + 回放对比（核心，非可选）
孩子录音跟读后，系统播放标准音频供孩子自己对比，不做自动评分，判断对错交给孩子自己的耳朵

> 目标：强化记忆，不增加认知负担，不引入"对错评判"带来的心理压力。

---

# 8. Review System（复习机制）

采用简化遗忘曲线：Day 1（学习）/ Day 2（复习）/ Day 4（复习）/ Day 7（复习）

复习内容自动混合：单词 / 拼读 / 句子

## 8.1 静态HTML下的实现方式（新增，替代原方案）

不依赖后端或账号系统，采用**本地进度记录 + 内容自带复习**的组合：

- 用 `localStorage` 记录孩子当前学到第几课（例如 `lastLessonCompleted: 12`）
- 孩子打开页面时，自动定位到"下一课"，不需要孩子或家长手动找
- 每节课内容里，Review环节按"课程编号差值"固定嵌入该复习哪些内容（例如第12课自动复习第11课的高频词 + 第9课的拼读规律），不依赖真实日历日期，只依赖"课序"，这样即使孩子隔了几天没学，复习节奏也不会错位
- 不做"忘记登录/换设备丢进度"的容错处理——这是静态HTML的已知限制，先接受，后续版本再考虑云端同步

## 8.2 阶段性综合复习课（新增）

Day1/2/4/7的复习解决的是"单个词/音别遗忘"，但不够——还需要定期让孩子把碎片知识串起来用，否则容易学了就丢，或者只会认单个词不会组合运用。

- **每周末安排一节"综合复习课"**，不引入任何新知识点，只整合本周内容
- **每个Phase结束（Week 2/5/7末）安排一节更大范围的"阶段复习课"**，整合本阶段所有词族、拼读规律、句型
- 内容做法：不是简单重复原有单词/句子，而是把本周期内学过的词、拼读规律**重新组合**，生成孩子没见过的新词/新句子组合（例如本周学了 -at 和 -an 两个词族，综合课里用这些词拼出新句子，而不是照搬当课例句）
- 时长可以比普通课略长（约30分钟），因为不引入新概念，孩子的认知负担不会显著增加
- 在数据模型中单独标记课程类型（见第10节 `type` 字段），与常规课区分，方便后续内容组织和排课

---

# 9. Onboarding（新增，Day 0）

零基础孩子第一次打开需要有人告诉他怎么用，但只需要一次：

- 用 3~5 分钟教孩子：点哪里听音 / 怎么跟读和回放对比 / 怎么点"完成"进入下一课
- 之后孩子可以完全自主操作，不需要家长每天陪同
- 建议做成一个极简的"Day 0"引导页，而不是写在说明文档里让家长转述

# 10. Lesson Data Model（课程结构）

```json
{
  "lesson_id": 1,
  "type": "regular",
  "focus": "short_a",
  "words": ["cat", "hat", "bat"],
  "audio": {
    "cat": "audio/cat.mp3",
    "hat": "audio/hat.mp3",
    "bat": "audio/bat.mp3"
  },
  "sentences": ["A cat.", "The cat is big."],
  "review_lessons": [0],
  "game": ["listen_match", "drag_spell", "shadow_read"]
}
```

```json
{
  "lesson_id": 6,
  "type": "consolidation",
  "scope": "week",
  "source_lessons": [1, 2, 3, 4, 5],
  "words": ["cat", "hat", "bat", "fan", "can", "pan"],
  "sentences": ["A cat can fan.", "The bat is on the pan."],
  "game": ["listen_match", "drag_spell", "shadow_read"]
}
```

> `review_lessons` 字段记录常规课自动带出哪些历史课的内容，按"课序差值"配置（对应第8.1节的复习逻辑）。
> `type` 字段区分课程类型：`regular`（常规课）/ `consolidation`（综合复习课，对应第8.2节）。综合复习课额外带 `scope`（week/phase）和 `source_lessons`（本次整合覆盖的课程范围），内容以词/句的重新组合为主，不引入新知识点。

# 11. Success Criteria（成功标准）

8周结束后：

**基础能力**
- 能拼读 CVC 单词
- 能读简单句
- 不依赖中文翻译

**学校适配**
- 能听懂基础课堂英语
- 能快速适应三年级英语课

**学习习惯**
- 每天20分钟可持续学习
- 不依赖家长讲解，只需一次启动引导（见第9节）

---

# 12. V1范围边界（非常重要）

V1 不做：
- 语法系统
- 写作训练
- 长篇阅读
- 复杂对话
- 单词分类记忆（水果/动物）
- **自动判分系统**（新增）
- **打卡/激励可视化系统**（新增）
- **跨设备进度同步**（新增，静态HTML限制，后续版本再考虑）

---

# 13. 核心理念总结

> 英语不是学出来的，是"读出来的"。

本课程核心路径：**声音 → 拼读 → 阅读 → 课堂英语**

实现路径：**静态页面 + 本地进度记忆 + 音频参照，不做多余的复杂机制。**
